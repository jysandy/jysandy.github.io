Things I've built

Gradient
{{ with resources.Get "images/gradient4.png" }} {{ end }}
{{ with resources.Get "images/gradient2.png" }} {{ end }}
{{ with resources.Get "images/gradient1.png" }} {{ end }} {{ with resources.Get "images/gradient3.png" }} {{ end }}

Gradient is a work-in-progress game engine. It currently includes physics simulation with Jolt, PBR direct lighting and materials, HDR rendering with Narkowicz ACES tonemapping, shadow mapping with PCF, normal mapping, bloom and environment mapping.

sandtrace
{{ with resources.Get "images/sandtrace1.png" }} {{ end }}
{{ with resources.Get "images/sandtrace2.png" }} {{ end }} {{ with resources.Get "images/sandtrace3.png" }} {{ end }}

sandtrace is a non-real-time recursive ray-tracer written in C++. It runs completely on the CPU, with no GPU acceleration.
